## Descripción 
este trabajo es sobre una aplicacion para una bibloteca donde se administraran los libros, los prestamos, la devolucion de los libros y las multas por retrasos.
La bibloteca podra asministrar los libros disponibles y los libros prestados en el momento, podra ver los prestamos activos como las fechas que se presto el libro y cuantos dias se prestara el libro 
ademas la bibloteca podra ver que usuario no ha regresado el libro prestado y podra calcularle la multa dependiendo de distintos factores como lo son el valor del libro y los dias del retraso 

 
## como se realizo el parcial
el parcial lo realizamos con java el cual utilizamos clases, herencias, polimorfismo y distintos factores que lleva la programacion orientada a objetos ademas tenemos los botones, estilos y formatos que usamos para 
darle una mejor umagen al parcial
